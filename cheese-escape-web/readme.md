🧀 チーズクエスト（Cheese Quest）
ブラウザで遊べるシンプルなアクションゲームです。
チーズをゴールまで運びながら、迫ってくるネズミを回避・撃退します。

🎮 ゲーム概要
プレイヤーはチーズをドラッグしてゴールまで運びます。
しかし、ネズミがチーズを狙って近づいてくるため、うまく回避する必要があります。
チーズは壁をすり抜けられない
ネズミはチーズを追いかけてくる
ネズミはタップで倒すことが可能
無事ゴールまで運べばクリア

🕹 操作方法
PC
ドラッグ：チーズを移動
クリック：ネズミを倒す
スマホ / タブレット
指でドラッグ：チーズを移動
タップ：ネズミを倒す

🧩 ゲームの特徴
シンプルで直感的な操作
子どもでも遊べる設計
ドラッグ操作＋アクション要素
レスポンシブ対応（スマホOK）
